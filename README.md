# 💎 Aishwarya Jewellery House — Haute Joaillerie

Welcome to the digital storefront of **Aishwarya Jewellery**, an ultra-premium, cinematic, and responsive web application designed for elite luxury shopping experiences. Inspired by high-end brands like Tanishq, this interface reflects absolute purity, trust, and timeless craftsmanship.

## ✨ High-End Features Embedded
* **Live Gold Rate Ticker:** Real-time dynamic simulation of 24KT, 22KT Gold, and Platinum rates.
* **Luxury Mega Dropdown Menu:** Immersive multi-column navigation with featured collections.
* **Cinematic Hero Slider:** Automated marquee banners showcasing flagship heritage collections.
* **Virtual AI Try-On Simulator:** Interactive customer simulation for trying on necklaces and earrings.
* **VIP Concierge Modal Suite:** Fully responsive digital secure booking engine for private boutique tours and HD video calls.
* **The Vault (Shopping Bag Engine):** Micro-interaction driven wishlist and luxury bag cart system.

## 🎨 Design Philosophy & Aesthetics
* **Theme:** Matte Royal Wine Red (`#4A0E10`), Sovereign Gold (`#D4AF37`), and Satin Silk Off-White (`#FAF8F5`).
* **Typography:** Elegant *Cinzel* for royal branding elements and crisp *Inter* for elite readability.
* **Interactions:** Modern smooth CSS transformations and custom micro-animations for an expensive user experience.

## 🛠️ Built With
* **HTML5** — Semantic high-luxury architecture.
* **CSS3** — Custom properties, CSS grid, and high-fashion animations.
* **Vanilla JavaScript (ES6+)** — Dynamic carousel state machine and interactive modal logic.
* **FontAwesome v6** — Premium typography vector icons.
